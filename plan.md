## GitHub Commit Agreement

Both students agreed to the following GitHub practices at the start of the project:

- Each student commits after every working session on their own machine
- Minimum one commit per day worked on the project
- Each student commits their own work directly — no proxy commits
- Commit messages must clearly describe the change made
- Examples of good commit messages:
  - `Add firewall rule screenshots fig08 to fig11`
  - `Write network.md Task 4.1.2 section`
  - `Upload http_capture.pcap to captures folder`
